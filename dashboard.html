💀 Why This Samba Is Dangerous

🔴
CVE-2003-0201 — trans2open

Buffer overflow in Samba < 2.2.8. Gives unauthenticated remote root — zero credentials needed.
✗ Confirmed: Samba 2.2.1a is below 2.2.8 — this target is exploitable.

🟣
Null sessions

Old Samba allows anonymous login by default. Lists users, shares, and OS info for free.

🟣
Username enumeration

Samba leaks valid usernames on failed logins — makes brute-forcing much easier.
